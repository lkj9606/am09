ul은 unordered list의 약자이다.
사전적 의미는 중요하지 않다.
속성값은 3개, circle, disc, square가 있으나, 현재는 css에서 대부분 처리한다.
사용도는 layout 구성시 navigation, board, sitemap 등 다양한 목록형태에 사용된다.
중요도가 높은 layout block

ol은 ordered list의 약자이다.
위 ul과 동일 하지만, 우선순위가 있거나 번호기준 정렬이 필요 할 때 사용하는 layout block.

dl
definition list

dl 은 definition list 의 약자이다.
정의형 목록인데, 마찬가지로 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
중요한 것은 레이아웃을 구성할때, 3단 형태 박스의 리스트 를 만들어 낼 때 유용하게 사용하기도 한다. (웹표준에서 어긋남)
